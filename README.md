# Sigrid Peng Portfolio

歡迎來到我的作品集！這裡展示了我開發的各類數位與實體遊戲專案，涵蓋實境解謎、劇本殺、行動應用程式以及敘事導向的獨立遊戲。

## 專案分類

- **實境遊戲**: 結合現場景物、AR 技術與 Linebot 的沉浸式體驗（如《童話之約》）。
- **劇本殺 (LARP)**: 聚焦於懸疑推理與情感沉浸的多結局腳本創作（如《命懸一線》）。
- **行動應用**: 使用 Kotlin/Java 開發的 Android App，包含密室逃脫遊戲與極輕量工具（如《R8M Escape》、《Magic Pen Note》）。
- **其他專案**: 探索多輪迴敘事與溫馨解謎的獨立遊戲製作（如《拯救之命》、《她留下的訊息》）。

## 技術棧

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Mobile Development**: Kotlin, Java (Android)

## 本地開發

**準備工作:**  Node.js

1. **安裝依賴:**
   ```bash
   npm install
   ```
2. **啟動開發伺服器:**
   ```bash
   npm run dev
   ```
